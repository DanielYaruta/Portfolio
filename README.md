## О проекте

Минималистичный статический сайт на **HTML, CSS и vanilla JavaScript** — без фреймворков
и сборщиков. Фоновое изображение — «Водяные лилии» Моне (1906), которое задаёт всю
визуальную философию: важна не форма, а состояние.

## Почему без фреймворков

Сайт — одна статическая страница, и для такой задачи React/Vue/Tailwind были бы
избыточной зависимостью, а не пользой. Вместо этого здесь:

- ручная анимация на `<canvas>` (цветочная рамка рисуется один раз в offscreen-canvas,
  дальше — только CSS transform/opacity, без перерисовки каждый кадр)
- CSS-переменные для тем вместо JS-библиотек теминга
- ноль внешних JS-зависимостей — весь интерактив на vanilla JS в одном `script.js`

## Особенности

- 🎨 Акварельная эстетика — размытые градиенты, мягкие тени, градиентная рамка карточки
- 🌊 Анимированная рябь на воде поверх фона
- 🐸 Лягушачья стайка выпрыгивает на аватар при наведении: рассаживаются по ободку друг за
  другом, моргают вразнобой, а последняя запрыгивает на макушку остальным — и все вместе
  спрыгивают обратно в пруд, когда курсор уходит (позиции пересчитываются от фактического
  размера аватара, поэтому корректно работают и на мобильных)
- ☕ Переливающаяся иконка кофейной чашки рядом с «Java Backend Developer»
- 🫧 Мыльные пузыри при клике / тапе на аватар
- 💧 Едва заметный водяной след за курсором
- 🌙 Переключение светлой / тёмной темы с сохранением в `localStorage` (без вспышки темы
  при загрузке — тема применяется блокирующим скриптом до рендера)
- 🔤 Типографика: Cormorant Garamond (заголовки) + Inter (текст)
- 📱 Адаптивная вёрстка (мобильные устройства поддерживаются)
- 🔗 Meta description, favicon и Open Graph — для аккуратного превью ссылки в соцсетях
- ♿ Уважение к `prefers-reduced-motion`
